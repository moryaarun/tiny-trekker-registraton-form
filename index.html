📋 How to Set Up Your Google Form (5 min guide)
1
Go to forms.google.com → click "Blank" to start a new form.
2
Set title: "Tiny Trekking – Explorer Registration". Add your tagline as description.
3
Add fields: Child's Name (Short answer, Required) → Age → School Name → Parent Name → Phone Number → Email → Trek Selection (Multiple choice) → Medical Info → How did you hear us?
4
At the end, add a Description block (not a question) with your GPay UPI ID, amount, and instruction to send screenshot to WhatsApp.
5
Go to Settings → Presentation → Confirmation message and paste your payment instructions there too.
6
Click Send → Link icon → Copy link. Share this link on WhatsApp groups, Instagram bio, and school notice boards!
